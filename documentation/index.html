how to use parallax background?

You will need the following attribute in your element data-type="background" data-speed="{speed}"

From the actual file, it looks like this:

						<div class="quote-box quote-box-bg" data-type="background" data-speed="10">
							...
						</div>

The next thing is to set the css to be like this:

						.quote-box-bg {
							background:url('../images/blurred-city.jpg') 50% 0 no-repeat fixed;
							background-size:cover;
						}

The important part is to have 50% 0 no-repeat fixed for background attribute.

how to trigger fade animation when content is in view?

In order to trigger the animation, you have to add the following style data-animation="{animation}" and make sure you animate.css is loaded if you ever change anything.

The element should look like this:

						<div data-animation="bounce">
							...
						</div>

For a full list of animation, check out Animate.css Website
